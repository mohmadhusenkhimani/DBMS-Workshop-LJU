# Classwork

## Hospital Appointment Management System

### Description

A hospital administrator wants to digitize the appointment process. The current manual system often results in multiple patients receiving appointments for the same doctor at the same date and time. The system should manage patient records, doctor details, and appointments efficiently.

### Database Tables

- Patient
- Doctor
- Appointment

### SQL Concepts Covered

- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT INTO
- INNER JOIN
- GROUP BY
- COUNT()
- SUM()
- WHERE
- ORDER BY

### Reports Generated

- List of All Patients
- List of All Doctors
- Appointment Details
- Patient Appointment History
- Doctor-wise Appointment Count
- Today's Appointments
- Total Consultation Revenue
- Doctor Revenue Report

# Homework

## Cinema Multiplex Management System

### Description

A multiplex cinema wants to replace its manual booking process with a computerized system. The system manages movie details, screen information, show schedules, customers, and ticket bookings while generating reports such as revenue and most-watched movies.

### Database Tables

- Movie
- Screen
- MovieShow
- Customer
- Booking

### SQL Concepts Covered

- Database Design
- Table Relationships
- Foreign Keys
- Multiple Table JOINs
- Aggregate Functions
- GROUP BY
- ORDER BY
- SQL Report Generation

### Reports Generated

- Display All Movies
- Display Screen Details
- Display Movie Schedule
- Customer Booking Details
- Total Revenue
- Most Watched Movie
- Screen-wise Collection
- Today's Revenue
- Movie-wise Revenue
- Screen Capacity Report

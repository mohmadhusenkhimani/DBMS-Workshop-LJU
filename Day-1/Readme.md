# Day 1 - DBMS Workshop

Welcome to **Day 1** of the **DBMS Workshop** conducted during the **Master of Computer Applications (MCA)** at **Lok Jagruti University (LJU)**.

On Day 1, we learned the fundamentals of **Relational Database Management System (RDBMS)** by designing databases, creating tables, establishing relationships using primary and foreign keys, inserting records, and writing SQL queries to generate reports.

# Classwork

## Student Management System

### Description

A college wants to develop a **Student Management System** to manage student records, course details, and enrollments.

### Database Tables

- Student
- Course
- Enrollment

### SQL Concepts Covered

- CREATE DATABASE
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT INTO
- INNER JOIN
- GROUP BY
- COUNT()
- WHERE
- ORDER BY

### Reports Generated

- List of All Students
- Courses Enrolled by Each Student
- Students Scoring Above 80 Marks
- Total Number of Courses Taken by Each Student
- Marks Obtained in Each Course

# Homework

## Order Management System

### Description

An online shopping company wants to develop an **Order Management System** to manage customers, products, and customer orders.

### Database Tables

- Customer
- Product
- Orders

### SQL Concepts Covered

- Table Relationships
- JOIN Operations
- Aggregate Functions
- Customer Invoice Generation
- Sales Reports

### Reports Generated

- List of All Customers
- List of All Products
- Total Products Purchased by Each Customer
- Quantity Purchased by a Particular Customer
- Customer Invoice
- Customers Purchasing Above ₹5000
- Customer & Product Details
- Total Sales
- Most Expensive Product
- Total Orders by Each Customer
